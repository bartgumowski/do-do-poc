Authentication
01

Sign Up — New User

✓ Account created, app open
GOAL: Create a new account and enter the app
│
→ User opens do-do.app in browser
↳ [Auth screen] loads with 3 sign-in options
│
[ Which method? ]
│
PATH A — Google OAuth
→ Tap "Sign in with Google"
↳ Google consent popup opens
→ Select Google account from list
→ Tap "Allow"
↳ Popup closes, redirected back to app
↳ Email is auto-verified (Google accounts always verified)
→ App checks if onboarding was completed
[ First time? ]
YES → [Onboarding screen] shown → see Flow 05
NO → [Board] loads directly
│
PATH B — Apple Sign In
→ Tap "Sign in with Apple"
↳ Apple auth sheet appears
→ Authenticate with Face ID / Touch ID / password
[ Hide email? ]
YES → Apple generates relay email address
NO → Real email used
↳ Redirected back to app, auto-verified
[ First time? ]
YES → [Onboarding screen]
NO → [Board]
│
PATH C — Email + Password
→ Tap "Email" option (or just fills email field)
→ Type email address
→ Type password
→ Tap "Create account"
↳ Supabase creates account
↳ [Verify email screen] shown ("Check your inbox")
→ User opens email inbox
[ Email arrived? ]
YES → Taps verify link in email
NO → Checks spam folder
[ In spam? ]
YES → Marks as not spam, taps link
NO → Waits or taps "Resend email" (no resend button currently)
↳ Link opens app tab, session verified
[ First time? ]
YES → [Onboarding screen]
NO → [Board]
│
✓ GOAL REACHED — User authenticated, app open
